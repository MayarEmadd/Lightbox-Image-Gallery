# Donut Lightbox Gallery

A modern and interactive **image gallery** showcasing delicious donuts.  
Built with **HTML5, CSS3, Bootstrap, and JavaScript**, this project allows users to click on any image and view it in a stylish lightbox with navigation between images.


# Features

- Click any image to open in a lightbox
- Navigate between images (Next / Previous)
- Close button to exit the lightbox
- Smooth and clean UI design


# Technologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript (Vanilla JS)
